# Stellar Burgers — Юнит-тесты

Этот репозиторий содержит юнит-тесты для класса **Burger** веб-приложения Stellar Burgers. Проект выполнен в рамках дипломной работы по курсу QA.

## Цель
Покрыть юнит-тестами класс `Burger` на 100%.

## Инструменты
- JUnit 4
- Mockito
- Параметризация тестов
- JaCoCo (отчёт о покрытии)

## Реализовано
- Проверка добавления, удаления и перемещения ингредиентов
- Проверка расчёта стоимости бургера
- Проверка формирования чека
